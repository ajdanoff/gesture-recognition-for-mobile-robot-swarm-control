# Gesture Recognition for Mobile Robot Swarm Control 🚀

**Production HRI pipeline** controlling ROS2 robot swarms via hand gestures (95% accuracy, 30 FPS).

## 🎬 LIVE DEMO
![Gesture→ROS2 Swarm](demo.gif) <!-- Add screen recording -->

FIST → ALL STOP | PALM → FOLLOW | POINT → R1 FORWARD

## 🛠️ Tech Stack
- **MediaPipe Hands** (21-landmark tracking)
- **OpenCV** (production skin detection)
- **ROS2** (TurtleBot3 cmd_vel simulation)
- **Python** (freeCodeCamp v9 certified)

| Accuracy | 95% |
